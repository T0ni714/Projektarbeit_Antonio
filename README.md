# Wer wird Millionär – ESP32 Quiz

Ein Quizspiel auf Basis von *Wer wird Millionär* für den ESP32 mit Touchscreen und SD-Karte.

---

## Features
- Touchscreen Steuerung  
- Fragen aus SD-Karte (JSON)  
- Multiple-Choice Antworten  
- Einfache Quiz-Logik  

---

##  Hardware
- ESP32  
- ILI9488 Touchscreen  
- SD-Kartenmodul + SD-Karte
